## Why This Repo?

- **Learn by doing**: Writing implementations helps solidify understanding

- **Reference**: Quick lookup for common patterns and solutions
- **Practice**: Useful for interview preparation and problem-solving

## How to Use

1. Browse through the topic folders
2. Each folder has implementations in both languages
3. Read the code and comments to understand the approach
4. Run the code locally to see it in action
5. Feel free to use these as reference for your own learning

## Running the Code

**Python:**

**JavaScript:**
```bash
node  topic-name/code.js
```
